# Stream notifier bot

Telegram-бот отправляет в выбранные группы уведомления о начале Twitch и
YouTube-эфиров. Каналы добавляются в личке с ботом, а группа для уведомлений
выбирается кнопкой.

## Переменные окружения

Скопируйте `.env.example` в секреты хостинга и заполните:

- `BOT_TOKEN` — токен от [BotFather](https://t.me/BotFather).
- `TWITCH_CLIENT_ID` и `TWITCH_CLIENT_SECRET` — приложение в [Twitch Developer Console](https://dev.twitch.tv/console/apps).
- `YOUTUBE_API_KEY` — ключ Google Cloud проекта с включённым YouTube Data API v3.

База SQLite хранится в `DB_PATH`. Для Docker смонтируйте `/app/data` как постоянный том, иначе состояние подписок будет потеряно при пересоздании контейнера.

## Настройка в Telegram

1. Добавьте бота администратором в нужный канал или группу. Для этого не нужно
   писать никаких команд в самом канале: бот автоматически запомнит чат в момент
   добавления.
2. Откройте личный чат с ботом и используйте появившееся меню:
   - **Добавить канал** — выбрать Twitch или YouTube, прислать ссылку и выбрать
     получателя уведомлений.
   - **Мои подписки** — открыть сведения о канале или удалить его с
     подтверждением.
   - **Проверить эфиры** — выполнить проверку немедленно.
   - **Оформление** — изменить заголовок уведомления, загрузить свою картинку
     или посмотреть текущие настройки.
3. При изменении заголовка доступна подстановка `{count}` — число новых эфиров.
4. Чтобы отменить незавершённый шаг, нажмите **Отмена** в нижнем меню.

Команды `/add`, `/chats`, `/list`, `/remove`, `/check`, `/template` и
`/preview` сохранены как запасной вариант.

Первый опрос после добавления запоминает текущий статус канала, но не уведомляет об уже идущем эфире. Уведомление придёт при следующем переходе канала из офлайна в эфир.

После обнаружения нового эфира бот ждёт 15 секунд и отправляет одно уведомление.
Если за это время начнётся другой эфир, он попадёт в то же сообщение. Если
второй эфир будет обнаружен позднее, бот отредактирует уже отправленное
уведомление, добавив информацию о нём. Уведомление содержит загруженную через
`/preview` картинку либо автоматическое превью первого эфира.

## Лимиты YouTube

Проверка YouTube использует `search.list` с фильтром `eventType=live`; этот вызов расходует API-квоту Google. Перед добавлением большого количества YouTube-каналов увеличьте `POLL_INTERVAL_SECONDS` и проверьте квоту проекта в Google Cloud Console.
